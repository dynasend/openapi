# Dynasend's OpenAPI Specification

This repository contains [OpenAPI specifications][openapi] for Dynasend's API.

[Changelog](https://github.com/dynasend/openapi/releases/)


Files can be found in the `openapi/` directory:

* `spec1.{json,yaml}:` OpenAPI 3.0 spec matching the public Dynasend API.
* `fixtures1.{json,yaml}`: Test fixtures for resources in spec3. See below for more information.

[openapi]: https://www.openapis.org/
